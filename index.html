About Us

NodeSchool is an open source project run by volunteers with two goals: to create high quality programming curriculum and to host community learning events.

What is Our Goal?

Join us and learn to code with us. You will learn the basics Node.js (and then some). If you already know JavaScript, it helps, but if not do not worry, we will help you in this matter too. Chances are good that you'll get further and understand more this time. :)
